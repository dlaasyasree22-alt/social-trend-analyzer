\# 📊 YouTube Trending Videos - Social Media Analysis



\## 🔍 Overview

Analysis of YouTube trending videos dataset to uncover patterns in views, engagement, upload timing, and channel performance across multiple countries.



\## 📁 Project Structure

\- `data/` - Raw dataset

\- `notebooks/` - Jupyter notebooks with full analysis

\- `visuals/` - Generated charts and graphs



\## 📈 Key Insights

\- 🏆 Top trending channels and their frequency

\- 📅 Best days and times to upload for trending

\- 🌍 Average views by country

\- ❤️ Engagement rate distribution

\- 👀 Most viewed trending videos



\## 🛠️ Tools Used

\- Python

\- Pandas \& NumPy

\- Matplotlib \& Seaborn

\- Jupyter Notebook



\## 🚀 How to Run

1\. Clone this repo

2\. Install dependencies: `pip install -r requirements.txt`

3\. Open `notebooks/` in Jupyter Notebook

4\. Run all cells!
